          ENS 2001: Intérprete de Ensamblador
          ===================================

-Historia de versiones:
10.03.2003 [Versión 1.0]
Se libera la versión 1.0. Nuevos ejecutables para Consola, Windows y
Linux. Nueva revisión del Manual de Usuario, corrigiendo algunas erratas.

31.12.2002 [Versión beta 0.9.7]
Se libera la revisión beta 0.9.7 que soluciona las incidencias detectadas
hasta el 31-12-2002. Nuevos ejecutables para Consola, Windows y Linux.

13.06.2002 [Versión beta 0.9.6]
Se libera la revisión beta 0.9.6 que soluciona las incidencias detectadas
hasta el 12-06-2002. Nuevos ejecutables para Consola y Windows.
Se pone a disposición la versión 0.9.6 para Linux.

24.05.2002 [Versión beta 0.9.5]
Se libera la revisión beta 0.9.5 que soluciona las incidencias detectadas
el 22-05-2002 y 23-05-2002. Nuevos ejecutables para Consola y Windows.

21.05.2002 [Versión beta 0.9.4]
Se libera la revisión beta 0.9.4 que soluciona la incidencia detectada el
20-05-2002. Nuevos ejecutables para Consola y Windows.

15.05.2002 [Versión beta 0.9.3]
Se libera la revisión beta 0.9.3 que soluciona la incidencia detectada el
14-05-2002. Nuevos ejecutables para Consola y Windows.

07.05.2002 [Versión beta 0.9.2]
Se libera la revisión beta 0.9.2 que soluciona la incidencia detectada el
06-05-2002. Nuevos ejecutables para Consola y Windows.
Añadida en el Manual de Usuario referencia al uso de caracteres especiales
en las cadenas de la pseudoinstrucción DATA.

30.04.2002 [Versión beta 0.9.1]
Se libera la revisión beta 0.9.1 que soluciona el bug detectado el
29-04-2002, y permite su ejecución en ausencia del archivo de biblioteca
de Borland vcl50.bpl. Nuevo ejecutable para Windows.

22.04.2002 [Versión beta 0.9]
Se pone a disposición la versión 0.9 beta para Consola y Windows.
